# User Flows

Tags: #flow #mvp #journey

Flow inti dari PRD revisi.

## Flow Surat

1. Warga login
2. Buka layanan surat
3. Pilih jenis surat
4. Isi keperluan dan data pendukung
5. Submit
6. Sistem buat tiket `PENDING`
7. Admin verifikasi
8. Admin proses
9. Sistem generate PDF
10. Status jadi `DONE` atau `REJECTED`
11. Warga lihat status dan unduh PDF

## Flow Pengaduan

1. Warga login
2. Buka menu pengaduan
3. Pilih kategori, isi judul dan deskripsi
4. Upload foto opsional
5. Sistem simpan `OPEN`
6. Admin triage
7. Admin ubah status sesuai progres
8. Warga pantau status dan catatan

## Flow Publikasi

1. Admin login
2. Buat berita/pengumuman
3. Simpan `DRAFT`
4. Review dan publish
5. Warga lihat konten publik
